# 家計簿アプリ
手軽に月の収入と支出を入力、管理することができるアプリです

### 開発の背景
会社の研修として、中途採用の未経験者４人が集まり２週間で家計簿アプリを作成しました。家計簿アプリを作ること、必要最低限の実装機能のみを課題としていただき、技術選定から実装までを自分たちで行いました。

## 使用技術
eclipse: version 4.38.0\
Java: version 21\

git\
github

## 技術選定理由
今回は全員が未経験ということで、できるだけこれまでに学習していたことが活かせる言語ということでJAVAを選びました。またGUIの部分では、一般的に広く使われている言語であれば、ネット上にも多くの情報があり、詰まった時にも解決策を探しやすいと予測し、フレームワークとしてswingを採用しています。\
データの保存系形式に関しては、今回JSONファイルを使用して保存する、という条件があったので、JacksonかGSONの二つが候補に挙がりましたが、今回のアプリではそこまで複雑な処理をしないアプリだったので、シンプルに使えるGSONを採用しました。

## 実装内容

### 月次サマリ（ホーム画面）
初期画面では今月の収入、支出、差額の合計がそれぞれ表示されています。こちらは下のプルダウンから日付を選ぶことで月ごとに表示が可能です。\
また直近３ヶ月の収支の合計が棒グラフとして表示されています。

### 入力画面
入力画面では日付、カテゴリ、金額、メモを入力することで、収入か収支として登録することができます。\
またカテゴリについては追加が可能です。

### 一覧画面
デフォルトの設定では、これまでに入力した収支の一覧が表示されています。\
希望の条件を入れることで特定の期間で絞り込みをかけて検索することも可能です。
