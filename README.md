# 家計簿アプリ
手軽に月の収入と支出を入力、管理することができるアプリです

### 開発の背景
会社の研修として、中途採用の未経験者４人が集まり２週間で家計簿アプリを作成しました。家計簿アプリを作ること、必要最低限の実装機能のみを課題としていただき、技術選定から、実装までを自分たちで行いました。

## 使用技術
eclipse: version 4.38.0\
java: version 21\
swing: version 1.9.0 \
g-son: version 2.10.1 \
git\
github

## 技術選定理由
今回は全員が未経験ということで、できるだけこれまでに学習していたことが活かせる言語ということでJAVAを選びました。またGUIの部分では一般的に広く使うわれている言語であれば、ネット上にも多くの情報があり、詰まった時にも解決策を探しやすいと予測し、FWとしてswingを採用しています。\
データの保存系形式に関しては、今回jsonファイルを使用して保存という条件があったので、jacksonかg-sonの二つが候補に挙がりましたが、今回のアプリではそこまで複雑な処理をしないつもりだったので、シンプルに使えるg-sonを採用しました。
